SMS / 카카오 알림톡 연동 준비

아직 실제 발송 기능은 연결하지 않았으며, 추후 연동을 위한 설정/정보 구조를 미리 정리해 둔 탭입니다.

1) 환경변수 설계 (예시)

  • SMS_PROVIDER: kakao / twilio / toast 등
  • SMS_API_KEY, SMS_API_SECRET
  • SMS_SENDER_ID: 발신 프로필/번호
  • KAKAO_ALIMTALK_SENDER_KEY, KAKAO_ALIMTALK_TEMPLATE_ID

2) 언제 발송할지 기준

  • 신규 신청 접수 시: 고객 안내용 / 대리점 알림용
  • 상태 변경 시: 접수완료 / 개통완료 / 정산완료 / 반려 전환 시 알림
  • 정산 안내: 월 단위 정산 금액 안내용

3) 템플릿 예시

상황 문자/알림톡 템플릿 예시
신규 신청 접수 "[MOSON] {{ 고객명 }}님, 신청이 정상 접수되었습니다. 담당자 확인 후 개별 연락드리겠습니다."
개통 완료 "[MOSON] {{ 통신사 }} {{ 상품 }} 개통이 완료되었습니다. 약정/요금/사은품 관련 문의는 언제든지 연락주세요."
정산 완료 "[MOSON] 이번 달 정산 {{ 금액 }}원이 입금되었습니다. 상세 내역은 대리점 어드민에서 확인 가능합니다."
반려 "[MOSON] 신청 건이 '{{ 반려사유 }}' 사유로 진행이 어려운 상태입니다. 필요 시 다시 문의 부탁드립니다."

  • 상담 상태 변경 로직(`admin_update_consult_status`)에서 이벤트 후킹
  • 발송 결과를 이메일 로그와 유사한 구조의 SMS 로그로 적재
  • 대리점별 발송 여부 On/Off 설정 (정산량이 적은 대리점은 비활성 등)
